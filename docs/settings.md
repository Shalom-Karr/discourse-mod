# Settings Reference

The plugin has a single setting, found under **Moderator Category Management** at `/admin/site_settings/category/discourse_mod_categories`.

## `mod_categories_enabled`

- **Default:** `false`
- **Client:** yes

Master switch for the plugin. When enabled, moderators (the built-in moderators group) can create, edit, and delete categories. When disabled, only admins can — standard core behavior.
